# InfoVault Library Management System

A console-based Library Management System built using Python.

## Features
- Add new books
- Register members
- Search books by category
- Issue books
- Update book details

## Technologies Used
- Python

## Project Structure
library-management-system
- main.py
- README.md

## How to Run
1. Download the project
2. Open terminal
3. Run:

python main.py
